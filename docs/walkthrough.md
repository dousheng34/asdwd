## 🚀 Как протестировать сайт прямо сейчас:

1. Перейдите по ссылке **[http://localhost:3000](http://localhost:3000)**.
2. Зарегистрируйте аккаунт (для удобства тестов по умолчанию в Supabase можно регистрироваться с любым паролем без реального подтверждения почты, если эта опция отключена в настройках).
3. Перейдите в Профиль, нажмите «Пополнить» и воспользуйтесь тестовым пополнением, чтобы добавить себе тенге (₸) на баланс.
4. Попробуйте создать объявление о продаже аккаунта.
